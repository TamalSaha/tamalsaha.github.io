Computer Adaptive Test

Computer Adaptive Test (CAT) System is an implementation of a form of computer-based test that adapts to the examinee's ability level. This form of test is usually used for TOEFL, GRE and GMAT exams.

Development Platform

This project was developed in Java using Borland JBuilder.

Asiatic Accounts Manager

This is a complete accounting package developed for one of the largest media consulting group in Bangladesh, ASIATIC MEDIA GROUP.

Main Functionalities:

  • Create Company
  • User Administration
  • Create, View, Edit, Close, Reopen Account
  • Bank Receive Voucher Entry
  • Bank Payment Voucher Entry
  • Cash Receive Voucher Entry
  • Cash Payment Voucher Entry
  • Journal Voucher Entry
  • Credit Note Entry
  • Debit Note Entry

Reports:

  • Specific Voucher Report
  • Specific Voucher Register Report
  • Bank Book
  • Cash Book
  • Cheque Register
  • Daily Cash and Bank Position Statement
  • Monthly Accountwise Report
  • Bank Reconciliation Report
  • General Ledger
  • Trial Balance
  • Trading Account
  • Profit and Loss Account
  • Appropriation P/L Account
  • Balance Sheet
  • Income Statement
  • Payment and Receive Statement
  • Cash Flow Statement
  • Notes to the Account

Benefits:

  • User friendly interface
  • Multiple companies and clients supported
  • Security for confidential information
  • Tax calculation
  • Client wise income statement
  • Employee wise expenditure report
  • Budget planning
  • Reports can be generated easily across accounting period
  • Reports can be exported to MS Excel, Pdf, html and other formats
  • Easy back up and recovery

Development Platform

This project was developed in C# 2.0 using Microsoft Visual Studio 2008. Oracle 9i was used as the backend database. Crystal Report was used for prepaing the reposrts.

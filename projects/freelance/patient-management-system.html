Patient Management System

Patient Management System (PMS) is a software to help doctors to manage their chamber and patients. It features a comprehensive waiting patients’ list management, setup of diagnosis, advices, diseases and surgery, database management, intelligent medication and multilanguage support. It can run seamlessly in multiple PCs or in a single PC.

The main features include:

  • Management of waiting &apm; prescribed patient’s list
  • Intelligent Setup to facilitate medication entry
  • Prescription in both Bangla and English
  • Research on patient’s personal and treatment information
  • The software can work in both networked or single mode
  • See patient’s full history
  • Intelligent search features
  • Prescription modification
  • Complete setup portions to customize your need
  • Special text in both Bangla and English
  • Database backup and restore facility
  • Automatic database backup
  • Ease of use

Development Platform

This project was developed in C# 2.0 using Microsoft Visual Studio 2008. SQL Server 2005 Express edition was used as the backend database. Crystal Report was used for prepaing the reposrts.
